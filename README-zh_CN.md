# 去他丫的考试

[English](./README.md) | 简体中文

## 介绍

是否在为刷题在苦恼？  

是否因为刷题形成肌肉记忆，脑子中默认记住了这个位置？  

是否因为刷题顺序固定，导致你全依赖顺序记忆背诵答案？  

没错，这个系统不能帮你考试。但是可以让你同样是刷题，但是刷的有质量！

## 特性

- 💡 TypeScript: 应用程序级 JavaScript 的语言
-  📜 卷子: 按卷子区分题组，创建自己的定制试卷
-  💎 组合：将不同卷子的问题，组合成新的卷子
-  📐 题目随机：题目顺序打乱随机
-  🚀 选项随机：答案选项打乱随机
